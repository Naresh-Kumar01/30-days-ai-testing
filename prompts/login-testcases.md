\# AI Prompt – Login Test Case Generator



\## Objective



Generate comprehensive test cases for the Login functionality.



\---



\## Prompt



Act as a Senior Quality Analyst with 10+ years of experience in Web Application Testing.



Generate comprehensive test cases for the Sign In functionality of an e-commerce application.



\### Feature Details



\- User signs in using Email and Password

\- Email must be valid

\- Password is case-sensitive

\- Remember Me checkbox

\- Forgot Password link

\- Active / Locked / Disabled account

\- Lock account after 5 failed attempts

\- Redirect to Dashboard after successful login



\### Cover



\- Positive

\- Negative

\- Boundary Value Analysis

\- Edge Cases

\- Validation

\- Security

\- UI

\- Accessibility

\- Usability



\### Output



Provide a Markdown table containing:



\- Test Case ID

\- Scenario

\- Preconditions

\- Test Steps

\- Test Data

\- Expected Result

\- Priority

\- Test Type



Generate at least 40 unique test cases.
