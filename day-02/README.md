\# 🏗 Prompt Formula



A well-structured prompt consists of:



```

Role

\+

Task

\+

Context

\+

Requirements

\+

Constraints

\+

Output Format

```

\# 🎯 Key Learnings



\- Prompt quality directly impacts AI output quality.

\- Clear instructions lead to more accurate and relevant responses.

\- Providing context improves AI understanding.

\- Specifying output format ensures structured results.

\- AI is a productivity tool that enhances, not replaces, Software Testing.
