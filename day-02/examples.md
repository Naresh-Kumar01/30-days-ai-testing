\# Day 02 – Prompt Engineering for Software Testers



\## Objective

Learn Prompt Engineering and how to communicate effectively with AI for Software Testing tasks.



\## Topics Covered

\- What is Prompt Engineering?

\- Prompt Structure

\- Role-based Prompting

\- Context Engineering

\- Constraints

\- Output Formatting



\## Practical Exercises

\- Test Case Prompt

\- Playwright Prompt

\- API Prompt

\- SQL Prompt



\## Key Takeaways

\- Better prompts produce better AI responses.

\- Always define the role, context, requirements, constraints, and output format.
